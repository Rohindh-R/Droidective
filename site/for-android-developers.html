For Android developers

Your adb workflow, without the terminal

Droidective is an all-in-one Android debugging tool for macOS. Tail logcat, watch performance, browse device files, and drive apps — every adb command behind a searchable ⌘K palette, so you stop memorising flags and start shipping.

Download for macOS View on GitHub

Free & open source · Apple Developer ID-signed & notarized · macOS 14+

The daily debugging loop, in one window

Live logcat

Stream device logs with instant search, level filters, and per-app scoping — find the stack trace without grepping a terminal.

Performance graphs

Live FPS, per-core CPU, RAM, and per-process usage while your build runs — catch jank and leaks at a glance.

File explorer

Browse, push, and pull files with real progress; full-filesystem access and read-write remount unlock on rooted devices.

App control

Force-stop, clear cache/data, grant or revoke permissions, inspect meminfo and the sandbox, and fire deep links per app.

Custom commands & ⌘K

Save your own adb one-liners and run anything from a Raycast-style command palette — recent commands are one keystroke away.

Device info & props

Read getprop, build/ABI, RAM, storage, battery, and root status in a single overview — no adb shell spelunking.
