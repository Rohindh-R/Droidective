Droidective / For support teams

For support & field engineers

See the device, pull the diagnostics, close the ticket

Droidective gives support teams a no-terminal way to diagnose Android devices on a Mac. Mirror the screen to see exactly what the user sees, pull a bug report and logs in one click, and check device health — no adb knowledge required.

Download for macOS View on GitHub

Free & open source · Apple Developer ID-signed & notarized · macOS 14+

Diagnose without the command line

Screen mirroring

Mirror the connected device live to reproduce what the user is reporting — over USB or Wi-Fi, scrcpy bundled in.

One-click bug report

Capture a full Android bug report and device state in a click to attach to the case — no copy-pasting from a shell.

Device health

Battery, storage, RAM, CPU, build, and root status in one overview, so you can triage hardware vs. software fast.

Pull the problem file

Grab a log, database, or config off the device with the file explorer — real progress, no sync-protocol guesswork.

Logcat with filters

Tail and filter logs to confirm an error reproduces, then export it straight into the ticket.

No terminal required

Everything is a click. Anyone on the team can run it — no adb commands, no SDK setup, just the app.

Faster triage, happier users
