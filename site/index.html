An entire adb toolkit, behind one keystroke

Stop memorizing flags and juggling terminal tabs. Droidective wraps the Android Debug Bridge in a fast, native macOS UI built for mobile and React Native developers.

⌘

Command palette

Fuzzy-search every tool and run it instantly. Pin favorites, assign global hotkeys, and never look up an adb flag again.

📱

Screen mirroring & control

A friendly GUI for scrcpy — mirror and control your device, tune bitrate, FPS and crop, and record straight to a file or GIF.

📜

Live logcat & crash catcher

Stream logs with level, tag and per-app filters, follow an app across restarts, and grab the last crash formatted for Slack or Jira.

🗂️

Device file explorer

Browse shared storage — or the whole filesystem on rooted devices — copy, move, delete, and push/pull files to your Mac.

📦

App management & inspection

Install, uninstall, force-stop, clear data, toggle runtime permissions, pull APKs, and browse a debug app's private sandbox.

📈

Performance monitor

Live per-core CPU, RAM, FPS & jank, and network throughput — charted, recordable, and exportable to JSON & CSV.

⚛️

React Native debugger tools

Open the dev menu, reload the JS bundle, reverse the Metro port, and point the app at any dev server — built for RN workflows.

📶

Wireless ADB & device control

Connect over Wi-Fi (including Android 11 pairing), manage Wi-Fi and private DNS, check root status, and flip developer restrictions.

State simulation

Fake the battery, force dark mode, change locale, scale fonts and density, set an HTTP proxy, and toggle networks to test edge cases.

One palette, every tool

A searchable sidebar groups all 43 tools into eight categories. Hit ⌘K from anywhere to jump straight to the one you need.

  • Pin favorites and assign global hotkeys
  • Persistent device bar — switch or target all devices
  • Per-feature command bar with the exact adb commands run

Enable only what you use

The feature catalog lets you toggle, reorder, and pin any of the 43 tools, so your sidebar stays exactly as busy as you want it.

  • Drag to reorder, or group by category
  • Setup Doctor checks your adb / scrcpy / emulator toolchain
  • Built-in terminal and command log

Whether you call it an adb tool, an all-in-one Android dev tool, a React Native debugger, or just the thing that saves you from typing adb shell a hundred times a day — Droidective puts every common Android debugging task behind a fast, native macOS interface. It's built for mobile developers, React Native engineers, and QA teams who live on the Android Debug Bridge.

Get started in a minute

Install Droidective

Ad-hoc signed and notarization-free — clearing quarantine once is all it takes.

1

Download the DMG

Grab the latest Droidective-vX.dmg from GitHub Releases and drag the app into Applications.

2

Clear quarantine once

It's not notarized (no paid Apple account), so run this once:

xattr -dr com.apple.quarantine /Applications/Droidective.app
3

Install adb

Droidective drives the Android Debug Bridge. If you don't have it:

brew install android-platform-tools
4

Open & connect

Launch the app — the built-in Doctor checks your toolchain, and you can connect over USB or Wi-Fi. Updates arrive automatically from then on.
